- [depolyer/depolyer 一个php部署工具](https://deployer.org/)
- [overtrue/laravel-lang 语言包](https://github.com/overtrue/laravel-lang)
- [dingo/api api包](https://github.com/dingo/api)
# 使用
```
git clone
composer install
cp .env.example .env
php artisan key:generate
php artisan jwt:secret
```
